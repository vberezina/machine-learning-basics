Вопросы и задания в билетах перемешаны случайным образом.

**Каждый студент должен знать краткие ответы на вопросы из блока [обязательный минимум](#Обязательный-минимум) и уметь произнести их вслух на экзамене (при необходимости).**

### Обязательный минимум

1. В чем заключается суть метода наименьших квадратов (МНК)?
   
3. Зачем в матрицу признаков добавляется столбец из единиц?
4. Что такое нормальное уравнение и когда его нельзя использовать?
5. Какую задачу решает логистическая регрессия и какой диапазон её значений?
6. Что такое «скорость обучения» (`learning rate`) в градиентном спуске?
7. В чем различие между недообучением (`underfitting`) и переобучением (`overfitting`)?
8. Как параметр $k$ в методе `LWLR` влияет на форму функции?


### Теоретические вопросы

1.	Основные задачи машинного обучения. Отличия обучения с учителем от обучения без учителя.

2.	Проблема переобучения и недообучения. Причины возникновения и способы визуальной диагностики.
3.	Компромисс между смещением и дисперсией. Влияние сложности модели на ошибку обобщения.
4.	Метрики качества классификации. Матрица ошибок и интерпретация вероятностных ответов.
5.	Метрики качества регрессии. Среднеквадратичная ошибка (`MSE`) и коэффициент детерминации.
6.	Методы валидации данных. Роль разбиения выборки на обучающую и тестовую части.
7.	Признаковое пространство. Необходимость и методы нормализации входных данных.
8.	Математическая модель логистической регрессии. Назначение и свойства функции сигмоиды.
9.	Решающая граница в классификации. Геометрическая интерпретация весов модели.
10.	Функция правдоподобия. Принцип максимизации правдоподобия при обучении классификатора.
11.	Логистическая функция потерь. Почему в задачах классификации не используется `MSE`.
12.	Фиктивный признак (`bias`). Влияние свободного члена на положение разделяющей гиперплоскости.
13.	Метод градиентного восхождения. Определение направления и величины шага обновления весов.
14.	Скорость обучения (`learning rate`). Последствия выбора слишком большого или малого шага.
15.	Пакетный градиентный спуск. Особенности работы со всей выборкой на каждой итерации.
16.	Стохастический градиентный спуск. Преимущества и недостатки при работе с большими данными.
17.	Адаптивный шаг обучения. Механизмы динамического изменения скорости обучения.
18.	Сходимость алгоритмов. Критерии остановки итерационного процесса оптимизации.
19.	Постановка задачи линейной регрессии. Аналитическое описание линейной зависимости.
20.	Метод наименьших квадратов. Математическая интерпретация минимизации остатков.
21.	Нормальное уравнение. Вывод формулы для нахождения глобального оптимума весов.
22.	Сингулярность матриц. Причины вырожденности матрицы признаков и способы решения проблемы.
23.	Матричная форма регрессии. Согласование размерностей векторов и матриц в вычислениях.
24.	Локально взвешенная линейная регрессия (`LWLR`). Концепция локальной аппроксимации.
25.	Гауссово ядро. Математический смысл весовой функции и зависимость от расстояния.
26.	Параметр сглаживания `k`. Влияние ширины ядра на гибкость аппроксимирующей кривой.
27.	Сравнение глобальной и локальной регрессии. Вычислительная сложность и область применения.
28.	Мультиколлинеарность. Влияние сильно коррелирующих признаков на стабильность решения.
29.	Регуляризация в регрессии. Общее представление о борьбе с переобучением через штрафы.
30.	Интерпретация весов. Физический и статистический смысл коэффициентов обученной модели.



### Практические задания

1.	Для набора данных из трех точек `(x=1, y=3)`, `(x=3, y=7)`, `(x=5, y=11)` сформируйте матрицу `X` с учетом фиктивного признака (bias) и вектор ответов `y` в матричном виде.

2.	Даны результаты промежуточных вычислений для линейной регрессии: матрица $(X^T * X) = [[2, 4], [4, 10]]$ и вектор $(X^T * y) = [[6], [16]]$. Вычислите вектор оптимальных весов `w` методом нормального уравнения.
3.	Модель линейной регрессии задана уравнением `y = 1.0 + 2.0x`. Вычислите значения прогнозов для выборки `x = [1, 2]` и рассчитайте для них величину среднеквадратичной ошибки (`MSE`), если реальные значения `y` равны `[3.5, 4.5]`.
4.	Линейная комбинация признаков для объекта (сумма произведений весов на признаки) составила `z = 1.0`. Рассчитайте вероятность принадлежности этого объекта к положительному классу, используя формулу сигмоиды (примите `e ≈ 2.72`).
5.	Даны веса обученного классификатора `w0 = -4` (смещение), `w1 = 1`, `w2 = 1`. Запишите уравнение прямой, разделяющей классы, и определите, какой класс модель присвоит объекту с признаками `(x1=2, x2=3)`.
6.	Выполняется прогноз для точки `x = 3.0`. В обучающей выборке имеется объект `x(i) = 3.2`. Рассчитайте вес этого объекта для локальной модели, если параметр сглаживания `k = 0.1` (используйте формулу Гауссова ядра).
7.	Текущее значение вектора весов составляет `w = [0.5, 0.5]`. Градиент функции потерь в этой точке равен `[0.2, -0.4]`, а скорость обучения (`alpha`) равна `0.1`. Рассчитайте значения весов после одного шага оптимизации.
8.	Дан вектор значений признака `x = [10, 20, 50]`. Выполните `Min-Max` нормализацию этих значений, приведя их к диапазону `[0, 1]`, и поясните необходимость этой процедуры для градиентных методов.
9.	При исследовании алгоритма `LWLR` были получены значения `MSE` на тестовой выборке для разных `k`: при `k=100` ошибка составила `15.2`, при `k=0.1` ошибка `2.1`, при `k=0.001` ошибка `12.8`. Определите, при каком значении `k` наблюдается эффект переобучения (`overfitting`).
10.	Имеется прямая регрессии `y = 5.0 - 1.5x`. Опишите характер изменения положения этой прямой на графике, если коэффициент наклона изменится с `-1.5` на `+2.0`, а свободный член уменьшится до `0`.
